Stage 1 — Discovery Layer Detail

How companies get registered and how new jobs are detected each day.

One-time Setup: ATS Auto-Detection

When the user runs python main.py add-company --name "Stripe" --slug stripe, the detector probes all 3 ATSes in parallel and stores the result.

User input
name + slug
→
Probe in parallel
Greenhouse · Lever · Ashby
boards-api.greenhouse.io
api.lever.co
api.ashbyhq.com
→
✓ 200 OK → save company + ATS type to SQLite
✗ All 404 → flag as unsupported, skip in daily run

Daily Run: New Job Detection

For each active company, fetch the full current job list from its ATS, diff against seen IDs in SQLite, emit only new ones downstream.
