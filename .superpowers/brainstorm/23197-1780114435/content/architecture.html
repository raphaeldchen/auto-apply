System Architecture

Four pipeline stages — stages 3 & 4 are future work, designed as plug-in slots from the start.

Trigger
⏰ Daily Scheduler
APScheduler — fires once per day, kicks off the pipeline
↓
Stage 1 — Discovery
ATS Poller
ATS Clients
Greenhouse · Lever · Ashby
Public job board APIs
Auto-Detector
Probes all 3 APIs at setup
Persists company → ATS mapping
Job Store
SQLite — seen job IDs
Emits only new postings
↓ new postings
Stage 2 — Filter
Relevance Pipeline
① Keyword Pre-filter
Title / level string match
Drops obvious mismatches cheaply
② LLM Scorer
Claude scores relevance 0–10
against desired role + level
↓ matched jobs

Resume Tailor
LLM rewrites resume sections to match each job description. Outputs a tailored PDF per application.
